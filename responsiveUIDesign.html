Responsive UI Design

Responsive Components

While it is important for a website to function correctly, it's also crucial that it looks good and works on multiple devices. This can be accomplished through a responsive layout that uses media queries to change the layout depending on the screen size. To practice this I have made a library of responsive components, based on layouts from various sources (including some layouts made during online courses i followed).

Used technologies: JavaScript, HTML, CSS
